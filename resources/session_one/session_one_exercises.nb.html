Basic Faceting

(Taken form R4DS)

  1. What happens when you facet a continuous variable?
  2. What do the empty cells in a plot with facet_grid(drv ~ cpl) mean? How do they relate to this plot?
ggplot(data = mpg) +

  1. What plots does the following code make? What does the . do?
ggplot(data = mpg) +
  geom_point(mapping = aes(x = displ, y = hwy)) +
  facet_grid(drv ~ .)

ggplot(data = mpg) +
  geom_point(mapping = aes(x = displ, y = hwy)) +
  facet_grid(. ~ cyl)
  1. Take the first faceted plot from the presentation:
ggplot(data = mpg) +
